Background and Features

Home Page.

After logging in with a Google Account (implemented with Google App Engine) the user is presented with a homepage and instructions on how to use this web app. This website features database interactivity - Submitting a task inserts data into the database. Clicking the trash icons calls a delete on the databases. Dragging and dropping tasks between columns updates the database. Looking at archived tasks and incomplete tasks calls a select on the database. Website interacts with 3 tables (users table - to pull login info/ userID, user task table - to pull out to do list items, and a user notes table- to store the notes associated with a task)

Submitting and modifying a todo list task information.

This website features database interactivity via PHPMyAdmin and MySQL. Submitting a task inserts data into the database. Clicking the trash icons calls a delete on the database. Dragging and dropping tasks between columns updates the database. Looking at archived tasks and incomplete tasks calls a select on the database. Additionally, AJAX is for updating the status of the task, updating the % completed task bar, removing deleted items from the screen. Third party UI widgets from Boostrap and JQuery were used.

Viewing All Incomplete Tasks.

Webpage pulls information regarding a user's incomplete tasks from the databases and populates the table accordingly. User session information is managed via HTML local storage objects.
