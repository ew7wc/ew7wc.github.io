Alarmtoed is an alarm and to-do list organizer.

This app allows the user to create task items, edit, and delete them easily. It also allows integration with a Raspberry Pi and LED lights to enable easy customization of flashing lights alerting the user of a task that's due! Additionally, the app provides Twitter integration, which posts when the user has completed a task. Optimized for 7 inch tablets.

At a Glance:

Sensor usage

Shake to turn off alarms (or just tap a button).

Twitter Integration

An option for the user to automatically post a message to Twitter when he finishes a task (corresponds to turning the alarm off by shaking the device or by pressing a button).

Raspberry Pi Integration

When the alarm goes off, the LED lights light up a specific color according to the task category that was set by the user.

Sound notification

Customize the ringtone or device vibration settings.

Alarmtoed is a task-oriented personal scheduler that alerts the user when his task is due. The user can add a task and customize its description with a name, a time, a date, a category, a priority, and some notes. The user must select a time, date, category and priority for each task. When a task is due, an alarm will go off! A customized ringtone will alert the user as well as several other things: the LED strip connected to the Raspberry Pi will light up a specific color corresponding to the type of task selected so the user does not have to manually check the app to know what he has to do. The intensity of the lights will correspond to the level of priority determined by the user. The user can shake the device or press a button to turn off the alarm. Upon dismissing the alarm, a message to Twitter will be sent, if the user configured the alarm to do so.
